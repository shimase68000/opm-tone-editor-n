# OPM Tone Editor 'N' v1.20 Release Notes

---
[English](RELEASES.md) | [日本語](RELEASES.ja.md)

## 新機能

- **slotmask 対応**
  - OPM パラメータテーブルの **SM**（Slotmask）が有効になり、KEYON 時に ON にするオペレータを音色ごとに指定可能です。

- **OPM コピー UNDO**
  - △△△ / ▽▽▽ による OPM パラメータテーブル間のコピーを 1 段階 Undo できます。
  - コピー後に表示される **[UNDO]** ボタンで元の状態に戻せます。

- **強制ロック（CTRL + !）**
  - 編集済みのファイルや音色でも **CTRL + !** でロックできるようになりました。
  - 強制ロック状態は `!` マーク（黄色）で表示されます。

- **読み込み時の自動選択（auto_select_imported_tones）**
  - MML / OPM / ZMS / SND / MDX を読み込んだ際に、音色を自動的に選択状態にできます。
  - `oe.jsn` でフォーマットごとに個別設定が可能です。

- **マウスオーバー説明表示（mouseover_help）**
  - UI 要素にマウスを乗せると説明テキストを表示する機能ですが、`oe.jsn` で有効 / 無効を設定可能です。（default: true）

## バグ修正

- **OPM 形式ファイル出力の修正**
  - AMS を出力すべき箇所に AMD が重複して出力されていたバグを修正しました。

- **ノートオフセット既定値の修正**
  - X68000 の OPM Clock（4.00MHz）に合わせ、既定値を **+48 → +45** に修正しました。

## 改善

- **POLY 数自動補正の改善**
  - OPM チャンネル数を増減する際、増加方向でも POLY 数が正しく補正されるようになりました。

- **アルゴリズム記号（OTG表記）の改善**
  - 自己フィードバックを持つ op1 のみ **bold** 表示になりました。

- **ロック / 選択操作の挙動修正**
  - ロック / アンロック操作では、音色編集済みマーク `*` は付かなくなりました。
  - 選択 / 選択解除操作では、音色編集済みマーク `*` は付かなくなりました。
  - 上記操作により、ファイル編集済みマーク `*` が付きます。

- **設定ファイル（oe.jsn）の拡張**
  - `scalekey.polyphony`：POLY 数の初期値を設定できます。
  - `auto_select_imported_tones`：読み込み時の自動選択をフォーマット別に設定できます。
  - `mouseover_help`：マウスオーバー説明表示の有効 / 無効を設定できます。
  - `true` / `false`、`"enable"` / `"disable"` 等を真偽値として扱えるようになりました。

## 動作要件

- 演奏機能を使用する場合は scalekey **v1.01 以降**が必要です。
